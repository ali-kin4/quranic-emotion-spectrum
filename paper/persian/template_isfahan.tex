% English-abstract environment: forces LTR direction, EB Garamond body font,
% Latin digits, and Western punctuation regardless of the surrounding xepersian
% RTL flow. Uses xepersian's \setlatintextfont mechanism via \textlatin to
% defeat the global digit-substitution that otherwise renders Latin numerals
% as missing-glyph boxes inside xepersian-controlled pages.
%
% Use as `\begin{englishabstract}...\end{englishabstract}`.
% Pre-declare a Latin font family so we can switch to it cleanly inside
% the english abstract; this avoids inline \fontspec invocation that
% xepersian's tokeniser sometimes mis-handles when followed by [options].

  {%
